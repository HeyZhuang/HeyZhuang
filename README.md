## 💼 Experience Highlights @ BlueFocus

在蓝色光标实习期间，我深度参与了营销科技领域的 AI 变革，核心攻坚以下方向：

* **🤖 Agent 架构与复杂工作流落地**：基于 LangChain、LangGraph 构建智能体核心逻辑，从零到一落地能够自主规划、调用工具解决复杂营销场景的 Agent 工作流。
* **🔍 企业级多模态 RAG 与检索优化**：利用 Google Vertex AI (`multimodalembedding@001`) 优化图像特征向量提取，结合 Weaviate、FAISS 及 pgvector 等向量数据库，搭建高效的企业级多模态搜索链路。
* **⚙️ 生产级异步流水线重构**：将原有的 Celery 异步任务系统平滑迁移至 Google Cloud Tasks，并结合 Cloud Run、GCS 和 FFmpeg，构建了高可用、高并发的异步视频封面生成及媒体处理服务。
* **🚀 AI 中间页与搜广投赋能**：将大模型技术深度融入广告搜索与投放链路，通过数据洞察与前端工程化，实现高转化率营销中间页的自动化生成与智能排版。
